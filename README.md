# Acerca del proyecto
Este proyecto consiste en una sencilla aplicación para proponer nombres y votar
entre ellos para seleccionar el favorito de acuerdo al número de votos
obtenidos.

Aunque existen diversas soluciones y herramientas similares, la razón principal
para la realización de este proyecto fue la de conocer y entender mejor diversos
frameworks de desarrollo web como *Laravel*, *Vue* y *Bootstrap*.
